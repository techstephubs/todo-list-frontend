# todo-list-frontend

Frontend em Angular do projeto Todo List.

## Pré-requisitos

- Node.js 18+
- npm

## Desenvolvimento

```bash
npm install
npm start
```

Acesse http://localhost:4200

## Build

```bash
npm run build
```

## Branches

- **main** – produção
- **develop** – desenvolvimento
